# site-to-pdf

```cd site-to-pdf```

## Setup venv
```python3 -m venv venv```  
```source venv/bin/activate```  

## Install requirements  
```pip install requests Pillow beautifulsoup4 playwright```  
```playwright install```  

## run script
```python3 site-to-pdf.py https://example.com```  

## deactivate venv
```deactivate```
